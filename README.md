\# Online Bookstore – PahanaEdu 📚



This is a Java-based online bookstore web application built using \*\*Servlets\*\*, \*\*JSP\*\*, and \*\*MySQL\*\*. The system follows a layered architecture and uses design patterns and SOLID principles.



---



\## 👤 User Roles



\- ✅ Admin

\- ✅ Staff

\- ✅ Registered Customer

\- ✅ Unregistered Customer (guest)



---



\## 🔐 Registration \& OTP Verification



\- Passwords hashed using SHA-256

\- OTP sent via email (via `OtpSenderService`)

\- OTP stored in `otp\_tokens` table

\- OTP verification required to activate account
